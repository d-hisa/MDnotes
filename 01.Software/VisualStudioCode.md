# Visual Studio code
## Installation

## Settings
### Use FiraCode
#### Install font
##### Mac
```bash
$ brew tap caskroom/fonts
$ brew cask install font-fira-code
```
#### Settings
<kbd>Cmd + , </kbd>で`setting.json`でを開き、下記を追加
```json
"editor.fontFamily": "Fira Code",
"editor.fontSize": 14,
"editor.fontLigatures": true
```
