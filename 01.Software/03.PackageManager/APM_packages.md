# apm Packages
## Commons
### Installation
```bash
# 目的のパッケージの確認
$ apm search PACKAGE_NAME
# パッケージのインストール
$ apm install PACKAGE_NAME
```
## atom-csv-markdown
```bash
atom-csv-markdown
```
### Description
CSV形式で書いたテーブルをMarkdwonテーブルへ変換してくれるパッケージ
### Usage
```
First Header, SecondeHeader
11,21
12,22
```
を変換する際は、カーソルで当該テキストを選択し、
右クリック->`Convert CSV to Markdwon table`
