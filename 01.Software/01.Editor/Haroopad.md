## settings
### フォント設定
#### エディタのフォント
[ファイル]->[設定]->[エディタ]->[default]を`Edit`
```css
~
editor {
    font-family: '任意のフォント', monospace !important;
}
~
```
とする。
#### ビューワのフォント
[ファイル]->[設定]->[ビューワ]->[default]を`Edit`
```css
~
>* {
    font-family:'任意のフォント', sans-serif;
}
code {
    font-family: '任意のフォント', monospace;
    font-weight: normal;
}
~
```
とする。
